# portofolio-web
Website portofolio pribadi yang menampilkan profil singkat, keterampilan, proyek, dan sertifikat. Dibangun menggunakan HTML, CSS, Bootstrap 5, dengan desain responsif agar nyaman diakses di berbagai perangkat. Project ini bertujuan sebagai media pembelajaran sekaligus portofolio untuk melamar pekerjaan di bidang IT, Data, dan Web Development.
